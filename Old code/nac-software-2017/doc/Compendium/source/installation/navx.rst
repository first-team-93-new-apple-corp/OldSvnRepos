NavX-MXP
========

Sometimes, our robot has a NavX-MXP board, often used for its gyro.
In these cases, all software laptops must have a copy of the NavX-MXP java library.

`Download by clicking the latest build link, extract, and install. <http://www.pdocs.kauailabs.com/navx-mxp/software/roborio-libraries/java/>`_
