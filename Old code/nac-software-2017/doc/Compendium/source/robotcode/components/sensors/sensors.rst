Sensors
=======

Sensors are components on the robot that inform the robot code about the robot or its surroundings.

For example, an encoder can measure how far the robot has moved.

For more information on sensors and their uses, see `WPILib's documentation on sensors <https://wpilib.screenstepslive.com/s/4485/m/13809/l/599711-how-do-i-do-_______-selecting-the-right-sensor-for-the-job>`_.
